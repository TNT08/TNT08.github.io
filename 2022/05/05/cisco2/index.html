万能的Tab键 及? 需英文状态下—-尽量不简写

三个模式

name> 用户模式

enable 进入特权模式

disable

exit 退出

logout 退出

no ip domain-lookup 关闭域名解析 否则会出现假死

name# 特权模式

configure terminal 进入全局配置模式

show 显示

name(config)# 全局配置模式

命令和命令/参数 之间需要有空格

hostname 参数 （修改名称）

三个密码

console

name(config)#line console 0 (0为编号—为所有console编号)

name(config-line)# password 参数

name(config-line)#login 启用密码

exit 退出

disable 退出

用户进入特权

name(config)#enable password 参数 （明文密码）

name(config)#enable secret 参数 （密文密码）

name#show running-config （运行配置文件查看）

远程终端密码

name(config)#line vty 0 15 （第一条线路的编号为0 15为最后一条线路 共16条线路）

name(config-line)# password 参数

name(config-line)#login 启用密码

name(config)# interface vlan 1

name(config-if)#ip address ip地址 子网掩码

name(config-if)#no shutdown

PC telnet ip地址
